### Instalation

Download the .exe and ejecute it, then the game should be runing just fine :D

### Description

The game consist on trying to go as fast as possible without falling out of the circutit, you accelerate pressing SPACE, be sure to stop accelerating on the turns, specially on the tight ones. Otherwise you will fly away out of the circuit and get a permanent game over (Press R to retry the game) 
***
### Downloading the code

You can download the whole project as a .zip file from the unity repository clicking on the button that says "Code</>" and you'll have the whole unity project for you to edit and toy around with, you can also clone the github repository following this method:
1.On GitHub, navigate to the main page of the repository.
2.Above the list of files, click Code.
3.To clone and open the repository with GitHub Desktop, click Open with GitHub Desktop.
4.Follow the prompts in GitHub Desktop to complete the clone.

El juego consiste en intentar ir lo más rápido posible sin salirte del circuito. Acelera pulsando la barra espaciadora y asegúrate de detenerte en las curvas, especialmente en las cerradas. Si no, saldrás volando del circuito y perderás la partida (pulsa R para Reiniciar).
